**The pyramid (how the layers stack):**

- **Layer 2 — *this doc*:** the full academic evidence/teaching library (institutions, lectures, papers, guidelines, technique).
- **Layer 3 — apex (next):** the distilled, decision-point-specific, **source-cited KG nodes** the on-device model actually retrieves over — built *from* Layer 2, validated by a mentor neurosurgeon.

> Compiled by four parallel research passes (institutional / teaching / primary-literature / operative-technique). **Every URL here was returned by a real search or fetch** — none invented. Spot-verify any single citation before it goes in the pitch deck.

## ⚖️ Ingestion-licensing legend (read before building the KB)
Kyro's KB *redistributes* distilled content, so licensing matters:
- 🟢 **Open / public-domain / CC** — safe to distil into cited KB nodes (WHO, US-gov CPGs, PMC/BMC open-access, CC-BY).
- 🟡 **Free to read but copyrighted** — link & summarize, don't copy verbatim; fine for learning/UX grounding (LITFL, ACEP Now, StatPearls, TeachMeSurgery, Geeky Medics, institutional patient pages).
- 🔴 **Paywalled or gated** — cite as authority only; extract the guideline-grade *conclusion*, never the text (NEJM/Lancet trials, Rhoton book, paywalled journals).
